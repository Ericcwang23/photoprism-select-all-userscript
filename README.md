# PhotoPrism Select All Userscript
A Tampermonkey userscript that adds a **Select All button** to the PhotoPrism library browser.
This makes it easier to quickly select or deselect all photos for batch operations.
## Features
Adds a **Select All** button to the PhotoPrism library
Toggle select / deselect all items
Lightweight and simple
Works with PhotoPrism running on localhost
## Installation
1. Install **Tampermonkey**
https://www.tampermonkey.net/
2. Install the script
Download:
photoprism-select-all.user.js
Then open it with Tampermonkey to install.
